A boxed object to express address in ALSA Sequencer. A ALSASeqAddr is a boxed object to express address in ALSA Sequencer. The address consists of two parts; the numeric ID of client and port.

The object wraps struct snd_seq_addr in UAPI of Linux sound subsystem.
