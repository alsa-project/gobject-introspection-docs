HitakiEfwFrame

Description [src] 

struct HitakiEfwFrame {
  /* No available fields */

A boxed object for the content of transaction frame in Fireworks protocol. A HitakiEfwFrame expresses the content of transaction frame defined by Echo Audio for protocol in Fireworks board module.
