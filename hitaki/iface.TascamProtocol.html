Description [src] 

interface Hitaki.TascamProtocol : GObject.Object

An interface for protocol of TASCAM FireWire series. TASCAM FireWire series transfer image of device state by either isochronous or asynchronous packets. The HitakiTascamProtocol is an object interface for the image and the change of state in the TASCAM FireWire protocol.

Prerequisite

In order to implement TascamProtocol, your type must inherit from GObject.
