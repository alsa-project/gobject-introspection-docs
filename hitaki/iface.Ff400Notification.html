interface Hitaki.Ff400Notification : GObject.Object

An interface for notification from Fireface 400. RME Fireface 400 transfers quadlet message to registered address when operating hardware knob. The message contains the values of signal level and target channel. The interface is to parse the message and to emit GObject signal.

Prerequisite

In order to implement Ff400Notification, your type must inherit from GObject.

Instance methods

hitaki_ff400_notification_parse_message

Parse messages transmitted by Fireface 400, then emit HitakiFf400Notification::knob-control if the message contains knob control.

Signals

Hitaki.Ff400Notification::knob-control

Emitted when the notification includes knob control. The range of value is different depending on target.
