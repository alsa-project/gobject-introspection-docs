Description [src]

Emitted when the notification includes knob control. The range of value is different depending on target.

For signal level of microphone input:

  • 0: zero
  • 10: +10.0 dB
  • 11: +11.0 dB
  • 12: +12.0 dB
  • …
  • 63: +63.0 dB:
  • 64: +64.0 dB:
  • 65: +65.0 dB:

For signal level of line input:

  • 0: zero
  • 1: +0.5 dB
  • 2: +1.0 dB
  • 3: +1.5 dB
  • …
  • 34: +17.0 dB:
  • 35: +17.5 dB:
  • 36: +18.0 dB:

For signal level of any type of output:

  • 63: -infinite
  • 62: -58.0 dB
  • 61: -56.0 dB
  • 60: -54.0 dB
  • 59: -53.0 dB
  • 58: -52.0 dB
  • …
  • 7: -1.0 dB
  • 6: 0.0 dB
  • 5: 1.0 dB
  • …
  • 2: +4.0 dB
  • 1: +5.0 dB
  • 0: +6.0 dB.

The isochronous cycle at which the message was sent. The first element is lower three bits of second up to 7 and the second field is for cycle up to 7999.
 The array must have 2 elements.
